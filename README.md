# 맛보기 동화 · Tale Buffet

> 세계 명작 동화와 한국 전래동화를 한자리에서. 줄거리·주요 인물·장별 스토리와 교육뮤지컬 수업 아이디어까지 담은 이야기 데이터베이스.

**맛보기 동화(Tale Buffet)** 는 교실에서 곧바로 쓸 수 있도록 정리한 동화 자료 도구입니다. 저작권이 소멸한 퍼블릭 도메인 이야기 **54편**을 한곳에 모아, 각 작품의 줄거리·주요 인물·장별 스토리를 빠르게 훑어보고 교육뮤지컬 수업으로 이어 갈 수 있게 만들었습니다.

---

## ✨ 주요 기능

- **빠른 검색** — 제목·인물·주제·줄거리 어디에 있든 키워드로 즉시 찾기 (예: `호랑이`, `질투`, `마녀`)
- **출처 필터** — 한국 전래동화 / 그림 형제 / 안데르센 / 그 밖의 명작으로 골라 보기
- **작품 상세 카드** — 줄거리 요약, 주요 인물 소개, 장별 스토리
- **교육뮤지컬 수업 더하기** — 작품마다 바로 활용할 수 있는 수업·연출 아이디어 3가지
- **외부 자료 연결** — 작품마다 **위키백과·나무위키** 버튼으로 더 깊이 탐색
- **인쇄 / PDF 저장 · 텍스트 복사** — 수업 자료로 손쉽게 내보내기
- **이용 안내(도움말)** — 상단 물음표 버튼으로 사용법 확인
- **반응형 · 접근성** — 모바일 우선 레이아웃, 키보드 조작(ESC 닫기)과 포커스 가시화 지원

## 📚 수록 동화 (총 54편)

| 출처 | 편수 | 예시 |
|------|------|------|
| 한국 전래동화 | 23편 | 흥부와 놀부, 심청전, 콩쥐팥쥐, 별주부전, 바리데기, 의좋은 형제, 장화홍련전 … |
| 그림 형제 · 페로 | 13편 | 신데렐라, 백설공주, 헨젤과 그레텔, 라푼젤, 룸펠슈틸츠헨, 구두장이와 요정 … |
| 안데르센 | 7편 | 인어공주, 미운 오리 새끼, 성냥팔이 소녀, 눈의 여왕, 백조 왕자 … |
| 그 밖의 명작 (유럽·미국·천일야화) | 11편 | 피노키오, 오즈의 마법사, 이상한 나라의 앨리스, 알라딘, 골디락스와 곰 세 마리 … |

> 수록 작품은 모두 저작권이 소멸한 퍼블릭 도메인 이야기입니다. 줄거리·인물·장 구성은 여러 이본(異本) 가운데 널리 알려진 판본을 기준으로 교육용으로 정리했습니다.

## 🚀 사용 방법

별도의 설치나 빌드가 필요 없는 **단일 HTML 파일**입니다.

1. 이 저장소를 내려받거나 `index.html` 파일을 저장합니다.
2. 브라우저에서 `index.html` 을 엽니다.

바로 웹에 올리고 싶다면 `index.html` 하나만 정적 호스팅(GitHub Pages, Netlify 등)에 배포하면 됩니다.

## 🔗 외부 링크 안내

각 작품 상세 화면의 **위키백과 · 나무위키** 버튼은 작품 제목으로 해당 백과사전 문서를 새 창에서 엽니다. 문서가 없거나 제목이 다른 경우 백과사전의 검색 안내 화면으로 연결됩니다.
